## About

BuildPro Construction is a professional construction company website that showcases their services, projects, and expertise in the industry. The website features a modern design with smooth animations, responsive layout, and intuitive navigation.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Mobile Menu**: Collapsible navigation menu for mobile screens
- **Scroll Animations**: Elements fade in as you scroll down the page
- **Counter Animations**: Animated statistics on the homepage
- **Project Filtering**: Filter projects by category on the projects page
- **Contact Form**: Interactive contact form with validation
- **Smooth Scrolling**: Smooth transitions between sections
- **Back to Top Button**: Quick navigation to the top of the page
- **Modern UI**: Clean, professional design with a focus on user experience

## Technologies Used

- **HTML5**: Markup language for the website structure
- **CSS3**: Styling with custom properties, grid, and flexbox
- **JavaScript**: Interactive features and animations
- **Font Awesome**: Icon library for social media and UI elements
- **Google Fonts**: Montserrat and Source Sans Pro fonts
- **Unsplash**: High-quality images for the website

## Installation

1. Clone the repository to your local machine
2. Open the project folder
3. All files are ready to use - no build process required

## Usage

1. Open `index.html` in your web browser to view the homepage
2. Navigate through the menu to explore different sections:
   - **Home**: Overview of the company and services
   - **About**: Company history and mission
   - **Services**: Detailed information about services offered
   - **Projects**: Portfolio of completed projects
   - **Contact**: Contact form and company information
